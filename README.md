# VIBE CHECK 9000™

> the world's most scientifically inaccurate vibe detection system.

A cyberpunk-themed interactive web experience that scans your "vibe" through a series of absurd questions and delivers a delightfully unhinged personality diagnosis. Built with vanilla HTML, CSS, and JavaScript — no frameworks, just vibes.

## features

- **Animated cyberpunk UI** — neon grid backgrounds, scanlines, glitch effects, and floating particles
- **Multi-stage scanning experience** — fake loading sequence with rotating quantum chakra messages
- **5 absurd personality questions** — shuffled options, branching stat math
- **12+ unique vibe diagnoses** — from `FERAL BUT POLITE` to `RETIRED HIMBO`
- **4-stat profile system** — Chaos, Charisma, Cosmic, and (most importantly) Raccoon
- **Persistent history & achievements** — saved to localStorage
- **Shareable results** — copy to clipboard or download a "report"
- **Hidden easter eggs** — Konami code unlocks rave mode and a secret duck. Try typing `vibe` anywhere on the page.

## stack

- HTML5
- CSS3 (custom animations, no framework)
- Vanilla JS (no dependencies)
- Canvas API for particle effects

## run it locally

```bash
git clone https://github.com/Sebby1770/vibe-check-9000.git
cd vibe-check-9000
open index.html
```

That's it. No build step. No `npm install`. Just open the file and vibe.
